# Install online:   pip install -r requirements-agent.txt

# NOT in this file: cryptography is installed as a SEPARATE optional step by
# install.py (_install_cryptography_optional) from the bundled wheelhouse.
# It is only used for portable Fernet decryption of the bundled encrypted
# service key (core/app_config.py); the import is already guarded, so its
# absence just falls back to Manual Mode. Making it a hard requirement here
# would fail the ENTIRE offline install whenever the bundle's wheelhouse
# predates the wheel (source/binary drift) -- see the 2.10.1 regression.

# --- E2E browser automation (Playwright over CDP attach) ---

# install.py after the core deps succeed (it ships no Windows wheel in the
# bundled wheelhouse, and its browsers must be downloaded anyway).
# If the optional install fails or is skipped, the agent still boots and all
# non-E2E routes work; only the /e2e/* endpoints raise 503.
